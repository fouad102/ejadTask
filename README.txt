# Souq Test Automation Project

## Description:
This project automates the following test cases for the Souq website:
1. Sign Up (Success and Failure)
2. Login
3. Forget Password
4. Select a category and verify the item page.

## Prerequisites:
1. Java Development Kit (JDK) 8 or later
2. Maven
3. Chrome Browser and ChromeDriver
